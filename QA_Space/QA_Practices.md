# QA Best Practices

## Test Design
- Write atomic, independent test cases
- Maintain clear naming conventions
- Follow POM (Page Object Model) design for UI automation

## Coding Standards
- Follow PEP8 for Python
- Keep reusable code in helpers/utils

## Defect Management
- Log detailed defect descriptions with steps and screenshots
- Track defects until closure in Jira

## Reporting
- Maintain daily test execution summary
- Generate Allure or HTML reports automatically post-execution
